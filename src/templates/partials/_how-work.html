В качестве основного шаблонизатора мы используем nunjucks, это позволяет нам ускорить процесс написания разметки HTML и сделать поддержку кода удобней и универсальней. Для облегчения использования и расширяемости проекта мы используем технологию компонентного подхода, которая описана в правилах именования css селекторов по методологии БЭМ.

Twig

Для css используем препроцесор sass, он позволяет использовать функции недоступные в самом CSS, например, переменные, вложенности, миксины, наследование и другие приятные вещи, после компиляции он возвращает нам всем извесный CSS, но сделаный быстрее и удобней.

Используем модульный подход написания компонентов, с умом используем другие патерны. В основном используем jQuery, vanilla так же не проблема.

У нас большой опыт работы с анимацией, мы знаем все тайны отрисовки элементов браузером и как достичь максимального показателя FPS.

Для максимального нашего с Вами удобства сборки проекта используем связку gulp/webpack. Контроль версий git. Работаем со всеми популярными инструментами веб-дизайна.
